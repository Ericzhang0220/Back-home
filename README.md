# App Objective
Eric observed that many teenagers today experience varying degrees of psychological stress, emotional isolation, and a lack of meaningful emotional support. Academic pressure, social media influence, and family expectations often leave students feeling overwhelmed.
He began exploring whether technology could provide a safe, accessible space where users can experience emotional warmth and care after a long day. The goal is not to replace professional therapy, but to create a supportive digital environment that promotes relaxation and emotional reflection.
To develop a mobile platform that provides emotional comfort through:
    1. AI-driven supportive conversations
    2. Peer-written encouragement messages
    3. A customizable virtual “home” environment
    4. Soft background music to enhance relaxation
The platform aims to combine emotional AI with immersive design to create a sense of psychological safety.


# Home Screen

# Shop Screen
- Cosmetic decorations for the room
    - Currency: likes
        - Can earn by getting likes from other people
        - Daily login bonus
    - Furniture (table, bed, drawer)
    - Pet (cats, dogs, birds)
    - Plants
    - Objects (Radio/Music Player, Cup, etc)

# Setting Screen (Profile)
- Profile Information
- Set UI text size (small, medium, large)
- chart of activity last month
    - for each day, show a mood (emoji?)
    - shows happiness level (%)
- Account Settings (log out, delete account)
- Audio Settings (adjust music volume)

# Hall Screen
- Community post timeline where users can post and give each other likes

# Room Screen
- edittable room
    - can add or remove stuff from the shop
    - can change the weather
    - time is tied to the real life time
- Pets and Plants, when tapped on, play an animation
- tapping the bed gives you a bedtime message
- tapping the table opens a "table view" which is where you can interact with the radio
- tapping the radio is how you can set your own music
- tapping the window lets you open/close
- Sunlight shines rays through the window and the angle changes depending on the time of day
    ## Music Feature
    - Connect to their music library
    - By default, the app's background music is randomly chosen from a set collection, but if they click on the radio object they can set their own music.
